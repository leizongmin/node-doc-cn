小提示

  1. 带有橙色边框的段落是暂时还没有人翻译的，应该优先翻译这些段落
  2. 对于别人已经翻译过的段落，如果能认可改翻译结果，请点旁边的 [赞] 按钮；否则请点旁边的 [编辑] 按钮，修改为你理想的翻译结果，并点 [保存]
  3. 使用 Tab / Shift + Tab 可上下切换段落；使用 Alt + S / Shift + Alt + S 可上下切换未翻译的段落
  4. 本文档采用 Markdown 格式编写，翻译时也请保持原来的文档格式
